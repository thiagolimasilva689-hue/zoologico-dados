# Zoológico - Pipeline de Dados com Python + SQLite + CSV

Projeto de pipeline ETL que importa dados de animais de um arquivo CSV para um banco SQLite.

## Funcionalidades
- Importar animais de CSV para SQLite
- Consultar animais por estado emocional
- Filtrar por peso e idade
- Relatório com COUNT, AVG, mais velho, mais novo
- Ordenação com ORDER BY

## Tecnologias
- Python 3
- SQLite3
- CSV

## Como usar
1. Execute: `python zoologico.py`

## Animais cadastrados
Dragon (Teiú), gloria (Hipopótamo), jade (Iguana), Lindinha (Anta), Rainha (Tigre), Brutus (Jacaré), moises (Jabuti), odin (Corvo), ruivinha (Arara)
